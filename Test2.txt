git clone [url] - клонирование репозитория
git add . - Добавить все (. - Обозначает текущий каталог со всем содержимым можно указать конкретный файл)
git commit -m "mesage" - Создать комит
git push - отправить в репозиторий
git pull - Стянуть с репозиторя
git rn file_name - удалить из локального-текущего репозитория
git branch branch_name - создать ветку
git merge - сливает ветку с другой (нужно находится в текущей ветке)
git status - получить статус
git checkout - переключение веток (отредактировано)
